# BookMyRoute

A full-stack bus ticket booking application.

## Project Structure

```
BookMyRoute/
├── frontend/   # React + Vite + Tailwind CSS
└── backend/    # Spring Boot + Java
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Make sure to configure your database and environment variables in `backend/src/main/resources/application.properties` before running.
